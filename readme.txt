== Diretórios e Arquivos ==
bin -> binário (compilado no Mac OS X 10.11), imagem de entrada (cat.pgm) e imagem de saída (output.pgm)
include -> arquivos .h
obj -> arquivos objeto (.o)
src -> arquivos .cpp
Makefile -> makefile para facilitar a compilação

== Instruções para Compilação ==
Para compilar basta executar o comando "make"

== Utilização ==
Supondo que você esteja no diretório "bin", basta rodar o binário:
./main cat.pgm
./main cat.pgm x1,y1 x2,y2
